# MybatisCodeHelper-Pro 破解版

# 介绍

MyBatisCodeHelper-Pro是IDEA下的一个插件，功能类似mybatis plugin。

# 使用说明

在IDEA中本地安装插件

激活方法：IDEA顶部菜单：Tools -> MybatisCodeHelper -> Activation -> OfflineActivation，在offline key框输入

任意字符串
